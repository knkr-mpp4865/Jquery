JQuery load()

This function is used to load remote data directly into selected elements.

$(selector).load(url,data,callback)

Data: optional string sent to server with request.

Example

<!doctype html>
<html>
<head>
<meta charset="utf-8">
<title>JQuery Ajax demo1</title>
<style>
#content{color:#610B38;}
</style>
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.1.1/jquery.min.js"></script>
</head>

<body>
<div></div>
<div id="content">
	
</div>

<script>
window.onload=function(){
 $("#content").load("./content/Jquery.load.html");
}

</script>
</body>
</html>
